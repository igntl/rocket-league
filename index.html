تسجيل الفريق (2v2)

يرجى ملء البيانات بدقة. الحد الأقصى للمشاركين في قاعدة البيانات المؤقتة هو 64 لاعباً (32 فريقاً).

اللاعب الأول (قائد الفريق)

اللاعب الثاني

حالة المقاعد المحجوزة: 0 / 64 لاعب

قوانين وشروط البطولة

يرجى الالتزام التام بالقوانين لتجنب الاستبعاد الفوري

عدم التفجير (No Demolition)

يُمنع منعاً باتاً تفجير سيارات الخصم عمداً أثناء اللعب. أي تفجير متعمد قد يعرض الفريق لعقوبات أو خسارة الجولة.

نظام الحسم (3 - 0)

تنتهي المباراة وتُحسم النتيجة فوراً بمجرد وصول أي فريق لـ 3 انتصارات مقابل 0 للخصم.

الاستفسارات والدعم

لأي استفسار، اعتراض، أو مشكلة واجهتكم أثناء البطولة، يرجى التوجه فوراً لفتح تذكرة في سيرفر الديسكورد.
